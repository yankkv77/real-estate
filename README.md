# RealEstate
Welcome to My Real Estate Website! This is an ongoing project aimed at creating a platform where real estate agents can post houses for sale and default users can find realtors or houses that meet their requirements. The website is being developed using Django DRF (Django REST Framework), React, and Redux.

# Features 
1. User Registration and Authentication:
2. User Roles
3. House Listings
4. Realtor Search
5. Interactive User Interface


# Technologies Used
 * Python
 * Django DRF (Django REST Framework)
 * React
 * Redux
 * HTML/CSS
